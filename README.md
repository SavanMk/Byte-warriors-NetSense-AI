# 📌 Problem Statement

Most internet users rely on traditional speed test tools that only display numbers like:

* Download speed
* Upload speed
* Ping

But these numbers **do not answer real user questions**, such as:

* ❓ Can I play online games smoothly?
* ❓ Why is my video call lagging?
* ❓ Is my internet good for streaming?

👉 Non-technical users struggle to interpret these values, leading to confusion and poor decisions.

---

# 💡 Proposed Solution

NetSense AI transforms raw network metrics into **clear, human-friendly insights**.

Instead of showing just numbers, the system explains:

* 📊 What your network performance means
* 🎯 What activities your connection supports
* ⚠️ What issues exist in your network
* 🛠️ How to improve your connection

      ↓

      ↓

# 🖥️ Setup Instructions

## 🔧 Run Locally

```bash
git clone YOUR_REPO_URL
cd NetSense-AI
```

```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

```bash
pip install -r requirements.txt
```

Create `.env` file:

```text
GEMINI_API_KEY=your_api_key_here
```

Run the app:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
